1. Information We Collect

The Chzzk Timetable extension collects the following information:

Automatically Generated Information

  • Anonymous User ID: Unique identifier automatically generated through Firebase anonymous authentication
  • Timetable Upload Records: Information about timetables uploaded by users (image URL, channel ID, upload time)
  • Voting Records: Like/dislike voting history
  • Report Records: Records of reported incorrect timetables

Information We Don't Collect

  • Email addresses
  • Real names
  • Phone numbers
  • Other personally identifiable information

2. How We Use Information

Collected information is used only for:

  • Storing and displaying timetable data
  • Preventing duplicate voting
  • Managing and verifying incorrect timetables
  • Improving service quality

3. Data Storage and Security

  • All data is stored encrypted in Google Firebase Firestore
  • Firebase uses Google Cloud Platform's security infrastructure
  • User authentication information is stored only in the local browser

4. Information Sharing

  • Collected information is not shared with third parties
  • Not provided externally except when legally required

5. User Rights

Users have the following rights:

  • Right to Delete: Delete your uploaded timetables at any time
  • Right to Access: Access your account information through backup codes
  • Right to Withdraw: Stop using the service by removing the extension

6. Third-Party Services

This extension uses the following services:

  • Google Firebase: Data storage and authentication
    Firebase Privacy Policy
  • Chzzk API: Channel information lookup
    Naver Privacy Policy

7. Cookies and Tracking

This extension:

  • Does not use cookies
  • Does not track users
  • Does not display advertisements
