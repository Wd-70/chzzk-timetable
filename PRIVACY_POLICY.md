# 개인정보 처리방침 (Privacy Policy)

**최종 수정일:** 2025년 3월 16일
**버전:** 1.0.0

---

## 한국어

### 1. 수집하는 정보

**치지직 시간표** 확장 프로그램은 다음과 같은 정보를 수집합니다:

#### 자동으로 생성되는 정보
- **익명 사용자 ID**: Firebase 익명 인증을 통해 자동 생성되는 고유 식별자
- **시간표 업로드 기록**: 사용자가 업로드한 시간표 정보 (이미지 URL, 채널 ID, 업로드 시간)
- **투표 기록**: 좋아요/싫어요 투표 내역
- **신고 기록**: 잘못된 시간표 신고 내역

#### 수집하지 않는 정보
- 이메일 주소
- 실명
- 전화번호
- 기타 개인 식별 가능 정보

### 2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 시간표 데이터 저장 및 표시
- 중복 투표 방지
- 잘못된 시간표 관리 및 검증
- 서비스 품질 향상

### 3. 정보 저장 및 보안

- 모든 데이터는 **Google Firebase Firestore**에 암호화되어 저장됩니다
- Firebase는 Google Cloud Platform의 보안 인프라를 사용합니다
- 사용자 인증 정보는 로컬 브라우저에만 저장됩니다

### 4. 정보 공유

- 수집된 정보는 **제3자와 공유하지 않습니다**
- 법적 요구가 있는 경우를 제외하고 외부에 제공하지 않습니다

### 5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

- **삭제권**: 본인이 업로드한 시간표를 언제든 삭제할 수 있습니다
- **접근권**: 백업 코드를 통해 본인의 계정 정보에 접근할 수 있습니다
- **철회권**: 확장 프로그램을 제거하여 서비스 사용을 중단할 수 있습니다

### 6. 데이터 보관 기간

- 활성 시간표: 사용자가 삭제하거나 관리자가 제거할 때까지
- 삭제된 시간표: 소프트 삭제 후 관리자 승인 시 영구 삭제
- 투표 기록: 로컬 브라우저 캐시에 보관

### 7. 쿠키 및 추적 기술

이 확장 프로그램은:
- **쿠키를 사용하지 않습니다**
- **사용자 추적을 하지 않습니다**
- **광고를 표시하지 않습니다**

### 8. 제3자 서비스

이 확장 프로그램은 다음 서비스를 사용합니다:

- **Google Firebase**: 데이터 저장 및 인증
  - [Firebase 개인정보 처리방침](https://firebase.google.com/support/privacy)
- **Chzzk API**: 채널 정보 조회
  - [네이버 개인정보 처리방침](https://policy.naver.com/policy/privacy.html)

### 9. 아동 개인정보 보호

이 서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

### 10. 개인정보 처리방침 변경

개인정보 처리방침이 변경될 경우:
- 확장 프로그램 업데이트를 통해 공지
- 중요한 변경사항은 별도 안내

### 11. 문의

개인정보 처리방침에 대한 문의사항:

## English

### 1. Information We Collect

The **Chzzk Timetable** extension collects the following information:

#### Automatically Generated Information
- **Anonymous User ID**: Unique identifier automatically generated through Firebase anonymous authentication
- **Timetable Upload Records**: Information about timetables uploaded by users (image URL, channel ID, upload time)
- **Voting Records**: Like/dislike voting history
- **Report Records**: Records of reported incorrect timetables

#### Information We Don't Collect
- Email addresses
- Real names
- Phone numbers
- Other personally identifiable information

### 2. How We Use Information

Collected information is used only for:

- Storing and displaying timetable data
- Preventing duplicate voting
- Managing and verifying incorrect timetables
- Improving service quality

### 3. Data Storage and Security

- All data is stored encrypted in **Google Firebase Firestore**
- Firebase uses Google Cloud Platform's security infrastructure
- User authentication information is stored only in the local browser

### 4. Information Sharing

- Collected information is **not shared with third parties**
- Not provided externally except when legally required

### 5. User Rights

Users have the following rights:

- **Right to Delete**: Delete your uploaded timetables at any time
- **Right to Access**: Access your account information through backup codes
- **Right to Withdraw**: Stop using the service by removing the extension

### 6. Data Retention Period

- Active timetables: Until deleted by user or removed by administrator
- Deleted timetables: Permanently deleted upon admin approval after soft deletion
- Voting records: Stored in local browser cache

### 7. Cookies and Tracking Technologies

This extension:
- **Does not use cookies**
- **Does not track users**
- **Does not display advertisements**

### 8. Third-Party Services

This extension uses the following services:

- **Google Firebase**: Data storage and authentication
  - [Firebase Privacy Policy](https://firebase.google.com/support/privacy)
- **Chzzk API**: Channel information lookup
  - [Naver Privacy Policy](https://policy.naver.com/policy/privacy_en.html)

### 9. Children's Privacy

This service is not directed to children under 14 years of age and does not knowingly collect personal information from children.

### 10. Changes to Privacy Policy

If the privacy policy changes:
- Notification through extension updates
- Separate notice for significant changes

### 11. Contact

For questions about this privacy policy:
- **GitHub Issues**: https://github.com/Wd-70/chzzk-timetable/issues
- **Email**: kjk7052@gmail.com
