# 스크린샷 및 프로모션 이미지 가이드

## 📸 필수 스크린샷 (Required Screenshots)

Chrome 웹스토어에 필요한 스크린샷을 준비하는 가이드입니다.

### 요구사항
- **최소 개수**: 1개 (권장: 3-5개)
- **크기**: 1280x800 또는 640x400
- **형식**: PNG 또는 JPG
- **최대 용량**: 각 5MB

---

## 📋 권장 스크린샷 목록

### 1. 메인 화면 - 시간표 보기
**파일명**: `screenshot-1-main.png`

**촬영 방법**:
1. 치지직 스트리머 페이지(예: https://chzzk.naver.com/live/xxxxx)로 이동
2. 확장 프로그램 아이콘 클릭
3. 시간표가 표시된 상태에서 전체 팝업 화면 캡처

**강조할 요소**:
- 채널 이름 및 ID
- 주차 정보 (날짜 범위)
- 시간표 이미지
- 좋아요/싫어요 카운트

**설명 예시**:
```
한국어: 스트리머의 주간 방송 시간표를 한눈에 확인하세요
영어: View weekly broadcast schedules at a glance
```

---

### 2. 주차 네비게이션
**파일명**: `screenshot-2-navigation.png`

**촬영 방법**:
1. 메인 화면에서 이전/다음 주 버튼이 잘 보이도록 캡처
2. 또는 다른 주차로 이동한 화면 캡처

**강조할 요소**:
- 주차 네비게이션 버튼 (◀ ▶)
- 현재 선택된 주차 정보
- 변경된 시간표

**설명 예시**:
```
한국어: 이전/다음 주 시간표도 쉽게 확인할 수 있습니다
영어: Easily navigate between different weeks
```

---

### 3. 투표 시스템
**파일명**: `screenshot-3-voting.png`

**촬영 방법**:
1. 좋아요 또는 싫어요 버튼을 클릭한 상태 (초록색/빨간색 강조)
2. 여러 시간표가 있는 경우 점수가 표시된 썸네일 영역 포함

**강조할 요소**:
- 좋아요/싫어요 버튼 (활성화 상태)
- 투표 수
- 썸네일의 점수 표시

**설명 예시**:
```
한국어: 커뮤니티 투표로 정확한 시간표를 검증하세요
영어: Community voting ensures accurate schedules
```

---

### 4. 시간표 등록
**파일명**: `screenshot-4-upload.png`

**촬영 방법**:
1. 스크롤을 내려 "시간표 등록하기" 섹션이 잘 보이도록 캡처
2. URL 입력란에 예시 URL 입력 (또는 빈 상태)

**강조할 요소**:
- 이미지 URL 입력란
- 등록하기 버튼
- 안내 텍스트

**설명 예시**:
```
한국어: 누구나 간편하게 시간표를 등록할 수 있습니다
영어: Anyone can easily upload new schedules
```

---

### 5. 안내 화면 (선택사항)
**파일명**: `screenshot-5-guide.png`

**촬영 방법**:

2. 안내 화면이 표시된 상태 캡처

**강조할 요소**:


## 🛠️ 스크린샷 촬영 도구

### 방법 1: 브라우저 개발자 도구
1. 확장 프로그램 팝업 열기
2. 팝업에서 우클릭 → "검사"
3. 개발자 도구 → Device Toolbar 활성화
4. 원하는 크기로 조정 (예: 400x600)
5. 스크린샷 캡처

### 방법 2: 스크린샷 도구
- Windows: Windows + Shift + S (Snipping Tool)
- Mac: Cmd + Shift + 4
- Chrome 확장: Awesome Screenshot, Nimbus Screenshot 등

### 방법 3: 온라인 리사이징
캡처한 스크린샷을 정확한 크기로 조정:
- https://www.iloveimg.com/resize-image
- https://www.img2go.com/resize-image

## 💡 팁

1. **일관성**: 모든 스크린샷에서 동일한 테마와 스타일 유지
2. **명확성**: 각 스크린샷은 하나의 주요 기능을 강조
3. **품질**: 고해상도 이미지 사용 (확대 시에도 깨끗함)
4. **순서**: 사용자 여정에 따라 논리적인 순서로 배치
   - 처음: 메인 기능 (시간표 보기)
   - 중간: 주요 기능들 (투표, 네비게이션)
   - 마지막: 등록 기능

5. **텍스트**: 각 스크린샷에 짧고 명확한 설명 추가
   - 한국어: 15-30자
   - 영어: 5-10 단어
