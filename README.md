# 치지직 시간표 확장 프로그램

치지직(Chzzk) 방송의 시간표를 간편하게 확인할 수 있는 크롬 확장 프로그램입니다.

## 주요 기능

- 각 방송마다 고유한 시간표 제공
- 사용자들이 시간표 이미지 URL 제보 가능
- 방송 페이지에 직접 시간표 보기 버튼 추가 (선택 기능)
- 방송인이 직접 시간표 등록 가능

## 설치 방법

1. 이 저장소를 다운로드 또는 클론합니다.
2. 크롬 브라우저에서 `chrome://extensions/` 페이지로 이동합니다.
3. 개발자 모드를 활성화합니다 (오른쪽 상단).
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다.
5. 다운로드한 저장소 폴더를 선택합니다.

## 사용 방법

### 시청자용

1. 치지직 방송을 시청하는 동안 확장 프로그램 아이콘을 클릭합니다.
2. 해당 방송에 등록된 시간표가 있다면 바로 확인할 수 있습니다.
3. 시간표가 없다면 직접 이미지 URL을 제보할 수 있습니다.
4. 설정에서 "방송 페이지에 시간표 버튼 표시" 옵션을 활성화하면 방송 페이지에 직접 시간표 보기 버튼이 표시됩니다.

### 방송인용

1. 확장 프로그램 팝업에서 "방송인도 직접 시간표를 등록할 수 있습니다" 링크를 클릭합니다.
2. 채널 ID와 시간표 이미지 URL을 입력하여 등록합니다.
3. 시간표 제작 가이드를 참고하여 효과적인 시간표를 제작할 수 있습니다.

## 향후 개발 계획

- 시간표 이미지 직접 업로드 기능
- 시간표 디자이너 도구 (방송인이 웹에서 직접 시간표 제작)
- 실시간 동기화 및 백업을 위한 서버 연동
- 다국어 지원

## 개인정보 및 데이터 처리

- 이 확장 프로그램은 사용자가 제보한 시간표 URL과 관련 정보만 저장합니다.
- 모든 데이터는 로컬 브라우저의 스토리지에만 저장됩니다.
- 현재 버전에서는 외부 서버로 데이터를 전송하지 않습니다.
